## 🔧 技术实现

### 主题管理
```typescript
const { isDarkMode, themeConfig, toggleTheme } = useTheme()
```

### 样式系统
- CSS 类名：`.arco-theme-dark`
- 主题属性：`arco-theme="dark"`
- 颜色变量：完整的 macOS 色彩系统

### 配置传递
```vue
<a-config-provider :theme="themeConfig">
  <!-- 所有 Arco 组件自动继承主题 -->
</a-config-provider>
```

## 🚀 使用说明

### 对于用户
1. 点击右上角或主页的"切换主题"按钮
2. 享受舒适的暗黑/亮色体验
3. 主题设置自动保存，下次打开生效

### 对于开发者
- 使用 `useTheme()` 组合式函数管理主题
- 通过 `.arco-theme-dark` 类名编写自定义暗黑样式
- 使用 `themeConfig` 传递主题配置到 Arco 组件
